🚀 Complete Setup Guide
Ek baar setup karo — phir lifetime kaam karega.
Part 1: Supabase (Free Database)
1
supabase.com pe account banao — GitHub se login karo (free)
2
New Project banao → Name: ca-crack → Region: Southeast Asia
3
Storage → New Bucket → Name: ca-materials → Public: ON
4
Project Settings → API → URL aur anon key copy karo
Part 2: Cloudflare Workers (Free API)
1
workers.cloudflare.com pe free account banao
2
Create Worker → worker.js ka code paste karo (file download karo)
3
KV Namespace banao → Name: CA_KV → Worker se bind karo
4
Environment Variables set karo:
ANTHROPIC_API_KEY = aapki API key
ADMIN_KEY = koi bhi strong password
5
Deploy karo → URL milegi jaise ca-crack.xyz.workers.dev
Part 3: App mein API URL set karo
1
App ka HTML file open karo (ca-crack-v2.html)
2
WORKER_URL variable mein apna Cloudflare URL daalein
3
File save karo — done! ✅
